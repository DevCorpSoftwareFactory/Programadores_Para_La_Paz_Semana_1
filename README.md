# Programadores por la Paz - Semana 1

> Participar en entornos digitales exige responsabilidad, organización y capacidad para documentar el trabajo realizado.

## Objetivo

Comprender la relación entre ciudadanía digital, uso responsable de la información, pensamiento algorítmico y organización técnica del trabajo, iniciando el uso de Git y GitHub para registrar y publicar la actividad realizada.

## Estructura del repositorio

```
semana1/
├── preguntas-semana1.txt    # Respuestas a preguntas de selección múltiple
├── reflexion-semana1.txt    # Reflexión sobre ciudadanía digital y herramientas técnicas
└── algoritmo-semana1.txt    # Algoritmo para compartir información confiable
```

## Contenido

| Archivo | Descripción |
|---------|-------------|
| `preguntas-semana1.txt` | Respuestas a las preguntas sobre ciudadanía digital, verificación de información y algoritmos |
| `reflexion-semana1.txt` | Reflexión de 8-10 líneas sobre el uso responsable de la información, participación ciudadana y organización del trabajo técnico |
| `algoritmo-semana1.txt` | Algoritmo de 8 pasos para compartir información confiable en internet |

## Herramientas utilizadas

- **Git** — Control de versiones para registrar cambios
- **GitHub** — Plataforma para publicar y compartir el trabajo
